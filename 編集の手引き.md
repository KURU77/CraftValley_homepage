# Craftvalley ホームページ かんたん編集マニュアル

プログラミングの知識がなくても、**商品・職人・お知らせ・イベント**を追加／修正できます。

---

## 1. 編集画面を開く

`admin.html` をダブルクリックして、ブラウザ（Chrome / Edge など）で開きます。

> ※ うまく画像が表示されないときは、フォルダごと簡易サーバーで開いてください。
> 　わからない場合は、まずはダブルクリックで開けばほぼ問題なく使えます。

---

## 2. 編集する

1. 上のタブで種類を選びます（🎁商品／🏔️職人／📰お知らせ／📅イベント）。
2. **「＋ 新規追加」** で新しく作る、または各項目の **「✏️ 編集」** で修正します。
3. 日本語・英語・画像などを入力します。
   - **日本語の欄（必須 ＊）** は必ず入力してください。
   - **英語の欄** は空でもOK（その場合、英語サイトには日本語が表示されます）。
     海外のお客様向けなので、できれば英語も入れるのがおすすめです。
   - **写真** は「ファイルを選択」から選ぶだけ。自動で軽量化して埋め込まれます。
4. **「この内容で保存」** を押します。

> この時点ではまだ「下書き」です（自分のパソコンにだけ保存されています）。

---

## 3. 仕上がりを確認する（プレビュー）

右上で確認したいデザイン（pop / wafuu など）を選び、**「👁 プレビュー」** を押すと、
実際のサイトの見た目で下書きを確認できます（画面下に「プレビュー表示中」と出ます）。

---

## 4. サイトに公開する

1. 右上の **「💾 data.js を書き出す」** を押します。
2. `data.js` というファイルがダウンロードされます。
3. そのファイルを、サイトのフォルダ内の

   ```
   shared / js / data.js
   ```

   に **上書き保存** してください（古い `data.js` と差し替える）。
4. サイトをアップロードしている場合は、この `data.js` をサーバーにアップすれば公開完了です。

これで6つのデザイン案すべてに、同じ内容が反映されます。

---

## よくある質問

**Q. 間違えて消してしまった／下書きをやり直したい**
→ **「公開データに戻す」** を押すと、いま公開中の `data.js` の状態に戻ります。

**Q. 写真は大きいサイズのままでいい？**
→ そのままでOKです。自動で幅1200pxほどに縮小して取り込みます。
　ただし枚数が非常に多いとファイルが重くなるため、不要な画像は削除してください。

**Q. 職人と商品のひもづけは？**
→ 商品の編集画面で「作り手（職人）」を選ぶだけ。職人ページに自動で表示されます。

---

## 編集してはいけないファイル（さわらないでください）

- `shared/js/cv-runtime.js` … 画像表示やプレビューの仕組み（プログラム）
- `各テーマ/js/main.js` … 各デザインの表示プログラム

更新するのは **`shared/js/data.js` だけ**（しかも編集画面が自動で作ってくれます）。
